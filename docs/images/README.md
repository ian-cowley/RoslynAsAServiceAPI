# Images and Screenshots

This folder contains images used in the documentation.

## Current Images

- `n8n-workflow-example.png` - Example n8n workflow using RoslynAsAServiceAPI
- `n8n-http-config.png` - HTTP Request node configuration in n8n

## Adding New Images

1. Take screenshots of your n8n workflows
2. Save them as PNG files in this folder
3. Reference them in the README using relative paths: `docs/images/filename.png`

## Image Guidelines

- Use PNG format for screenshots
- Keep file sizes reasonable (< 1MB when possible)
- Use descriptive filenames
- Include alt text in README references for accessibility
